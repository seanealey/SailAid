# SailAid
project page for a navigation aid for visually impaired sailors


Component List:

Boat Module:
- Heltec Wireless tracker v1.2 https://heltec.org/project/wireless-tracker/
- Antanna
- 5v battery
- button

Buoy Module:
-  Heltec Wireless tracker v1.2 https://heltec.org/project/wireless-tracker/
- buzzer (LA-8904) jaycar

Central Module:
- Heltec Wireless tracker v1.2 https://heltec.org/project/wireless-tracker/


Pins being used:

Boat:
    GPS:
    3, 33, 34, 35, 36
    LoRa:
    8,9,10,11,12,13,14
    Aux:
    - button 
    - gps fix led 45

Buoy:
    GPS:
    3, 33, 34, 35, 36
    LoRa:
    8,9,10,11,12,13,14
    Aux:
